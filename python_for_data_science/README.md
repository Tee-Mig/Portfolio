# python_for_data_science

## Data Oriented Design

Ce dépôt regroupe une série de petits modules Python orientés data science.  
L’objectif est de pratiquer :

- la manipulation de données numériques,
- les fonctions d’ordre supérieur et les closures,
- les décorateurs,
- les `dataclass` et la conception de structures de données. :contentReference[oaicite:0]{index=0}

L’ensemble du projet suit les contraintes de la piscine Python 42.

## Prérequis

- **Python** : 3.10 :contentReference[oaicite:1]{index=1}
- Pas de variables globales.
